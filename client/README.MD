HOW TO RUN THE PROGRAM:
============


1. install dependencies
2. open two clients/terminals
3. type `node index.js ws://localhost:3000 game-123` for the first server
4. type `node index.js ws://localhost:3001 game-123` for the second server

This will connect you to the same game with different servers

Redis DB details remain in code for purpose so you can check easily, the DB will be deleted later on.
